## ✨ Features
- **Fully responsive** — mobile-first design with breakpoints at 768px (tablet) and 1024px (desktop)
- **Semantic HTML5** — proper use of `header`, `nav`, `main`, `article`, and `footer` for accessibility and SEO
- **CSS Grid** for macro page layout (menu grid, location section)
- **Flexbox** for micro components (navigation bar)
- **Fluid typography** using `clamp()`
- **Mobile hamburger menu** with JavaScript toggle and ARIA states
- **Accessible** — WCAG-conscious color contrast, alt text on all images, ARIA labels
- **Performance-aware** — preloaded hero image (LCP), reserved image dimensions via `aspect-ratio` (CLS prevention), lazy-loaded menu images
- **Live Google Maps embed** for the location section
